eBay connection sandbox

Where do these come from?
  1. Create a (free) developer account at developer.ebay.com and an application keyset — sandbox and production are separate keysets.
  2. Client ID / Client Secret are the keyset's App ID and Cert ID (Application Keys page).
  3. Refresh token: on the Application Keys page click User Tokens → Get a User Token → OAuth → sign in with your eBay seller account and agree. Grant the sell.inventory and sell.account scopes.
  4. Save here, then Verify — LP-OS mints short-lived access tokens from the refresh token automatically.
